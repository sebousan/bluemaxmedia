# BMM

## Export Wordpress to Markdown

https://github.com/lonekorean/wordpress-export-to-markdown

```bash
npx wordpress-export-to-markdown --input=export.xml --post-folders=false --prefix-date=true --date-folders=year --include-time=true --save-images=attached --frontmatter-fields=title,date,categories:projects_categories,tags:projects_tags,coverImage:image,draft 
```
